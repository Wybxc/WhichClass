关于

Which Class 是 What Class 的精神继承者，支持按照时间和地点筛选课程。

数据来源

课程数据来自每学期初从选课网爬取。由于选课网显示的课程在每个人看来都不一样（比如已经选过的课程就不会出现在选课网上，有的专业课只有本院的人才能看到信息），所以会有很多课程信息缺失（课程补充功能正在开发中）。

为什么是一个新的项目？

What Class 是一个很好的项目，但是它的代码质量不高，维护起来很困难。（其实主要是因为我不会用关系型数据库orz）

Which Class 使用了新的技术栈，尤其是 HTMX + 服务端渲染的 No-JS 方案，更适合饱受前端框架之苦的宝宝们。此外，项目也已在 GitHub 开源，欢迎大家贡献代码。

项目使用的技术

  • Litestar & Jinja：Python ASGI Web 框架和模板引擎，实现服务端渲染
  • MongoDB：NoSQL 数据库，存储课程信息，实现索引查询和文本搜索
  • Tailwind CSS & Daisy UI：CSS 框架和组件库，支持高达35种不同主题自由切换
  • HTMX：AJAX 框架，使用 HTML 标签代替 JavaScript 进行交互
